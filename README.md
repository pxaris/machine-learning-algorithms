# Machine Learning Algorithms

- *Linear Regression* from scratch using Stochastic Gradient Descent
- *Polynomial Regression* studying different degrees of the fitting polynomial
- *k-means* using Elbow method to find the optimal number of clusters
- *Logistic Regression* on Spam detection task using tf-idf to vectorize the data
- *Principal Component Analysis* from scratch - testing its performance via a Logistic Regression classifier
